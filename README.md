# resumo-do-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO, localizando serviços por categoria


Na aula sobre Microsoft Azure, foram abordados os principais serviços oferecidos pela plataforma e como localizá-los por categoria. A interface é a mesma para usuários comuns e empresas, mas contas gratuitas têm limitações em certos serviços e regiões. O Azure permite várias customizações, como trocar idioma e ajustar o layout.

Os serviços são organizados por categorias, como Computação, Contêiners, Bancos de Dados, e DevOps, e, em algumas categorias, há subcategorias (por exemplo, em Computação: IaaS, PaaS, Servidores sem estado, entre outros). Alguns serviços estão em "Versão Prévia," indicando que ainda estão em teste e podem não ter suporte contínuo.

A plataforma é amigável, com navegação intuitiva e uma organização que facilita encontrar os serviços desejados.
